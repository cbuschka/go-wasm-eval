# Eval of wasm compiled go program

## Prerequisites

* Linux
* bash
* go installed
* GNU make installed
* python3 installed

## Usage

### Build

```
make build
```

### Start server

```
make serve
```

### Open in browser

Point your browser to [http://localhost:9000](http://localhost:9000).
